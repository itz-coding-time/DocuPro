üõ† Features

*Standardized Documentation: Generates HR-ready statements matching corporate DOCX schemas for immediate submission.

*Intelligent Roster Management: Specialized parser for "Gross" Sheetz Weekly Schedule exports that filters for night-shift associates automatically.

*The Ghost Protocol: Logic-based visibility that hides associates from the active roster once they have been sent home or dismissed for the shift.

*Shift-Aware Logic: Handles the unique "Time-Travel" math of overnight shifts (e.g., ensuring 2:00 AM Tuesday is correctly logged under Monday's shift).

*Local Persistence: Data is stored locally via GSON, ensuring your logs survive reboots without requiring cloud infrastructure.

üöÄ Getting Started

Prerequisites

Android Studio Ladybug or newer.

Java 17+.

A "Gross" .xlsx schedule export if using the automated importer.

Setup

1. Clone the repository: git clone <your-repo-url>

2. Open in Android Studio.

3. Allow Gradle to sync (this will download Apache POI for Excel handling and Gson for storage).

4. Run on your physical device or emulator.
